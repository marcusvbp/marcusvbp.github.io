## Português

Esta Política de Privacidade descreve como o aplicativo Julius ("Aplicativo") coleta, usa, armazena e protege as informações dos usuários. Ao usar o Aplicativo, você concorda com as práticas descritas neste documento.

### 1. Responsável pelo Tratamento de Dados

**Desenvolvedor:** Marcus Vinicius Barros Pontes
**E-mail de contato:** marcusviniciusbp@gmail.com

### 2. Dados Coletados

#### 2.1 Dados fornecidos pelo usuário
- Valores e descrições de gastos inseridos manualmente;
- Configurações de orçamento (limite mensal, despesas fixas, dia de corte);
- Preferências do Aplicativo (idioma, modo escuro, ativação de sincronização).

#### 2.2 Dados de conta (opcionais)
- Nome de exibição e endereço de e-mail da conta Google, quando o usuário opta por fazer login via Google Sign-In;
- Identificador único de usuário gerado pelo Firebase Authentication.

#### 2.3 Dados de uso e diagnóstico
- Eventos de uso anônimos coletados pelo Firebase Analytics para fins de melhoria do Aplicativo (ex.: telas visitadas, funcionalidades utilizadas);
- Relatórios de falhas e erros técnicos via Firebase Crashlytics.

#### 2.4 Dados de publicidade (somente versão gratuita)
- O Google AdMob pode coletar dados para exibição de anúncios personalizados, conforme a [Política de Privacidade do Google](https://policies.google.com/privacy). A assinatura Julius Pro desativa completamente a exibição de anúncios.

#### 2.5 Dados de assinatura
- Informações de compra e status de assinatura gerenciados pelo RevenueCat. O Aplicativo não armazena dados de cartão de crédito ou informações de pagamento — essas informações são processadas exclusivamente pela Apple App Store ou Google Play Store.

### 3. Finalidade do Tratamento

Os dados são coletados e utilizados para as seguintes finalidades:

| Finalidade | Base legal (LGPD) |
|---|---|
| Armazenar e exibir os gastos registrados pelo usuário | Execução de contrato |
| Sincronizar dados entre dispositivos (quando habilitado) | Consentimento |
| Autenticar o usuário e vincular dados à conta | Consentimento |
| Melhorar o Aplicativo com base em dados de uso anônimos | Legítimo interesse |
| Exibir anúncios (versão gratuita) | Consentimento |
| Gerenciar assinaturas Julius Pro | Execução de contrato |

### 4. Armazenamento de Dados

#### 4.1 Armazenamento local
Por padrão, todos os dados de gastos e configurações são armazenados exclusivamente no dispositivo do usuário, utilizando o armazenamento interno do sistema operacional. O Aplicativo não acessa a rede por padrão.

#### 4.2 Armazenamento na nuvem (opt-in)
Quando o usuário ativa a funcionalidade "Sincronização na Nuvem" nas configurações, os dados são enviados e armazenados no Firebase Firestore (Google LLC), com servidores localizados nos Estados Unidos. Essa funcionalidade pode ser desativada a qualquer momento, e os dados na nuvem podem ser excluídos mediante solicitação.

### 5. Compartilhamento de Dados

O Aplicativo não vende dados pessoais a terceiros. Os dados podem ser compartilhados apenas com os seguintes prestadores de serviço, estritamente para operação do Aplicativo:

| Serviço | Finalidade | Política de Privacidade |
|---|---|---|
| Firebase (Google LLC) | Autenticação, sincronização e analytics | [firebase.google.com/support/privacy](https://firebase.google.com/support/privacy) |
| Google AdMob | Exibição de anúncios (versão gratuita) | [policies.google.com/privacy](https://policies.google.com/privacy) |
| RevenueCat, Inc. | Gerenciamento de assinaturas | [revenuecat.com/privacy](https://www.revenuecat.com/privacy) |
| Apple App Store / Google Play | Processamento de pagamentos | Respectivas políticas das lojas |

### 6. Retenção de Dados

- **Dados locais:** permanecem no dispositivo até que o usuário desinstale o Aplicativo ou exclua os dados manualmente.
- **Dados na nuvem:** retidos enquanto a conta do usuário estiver ativa. Podem ser excluídos mediante solicitação ao e-mail de contato.
- **Dados de analytics:** retidos por até 14 meses, conforme a política padrão do Firebase Analytics.

### 7. Direitos do Usuário

Em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018), o usuário tem os seguintes direitos:

- **Acesso:** solicitar confirmação de que seus dados são tratados e obter uma cópia;
- **Correção:** solicitar a correção de dados incompletos, inexatos ou desatualizados;
- **Exclusão:** solicitar a exclusão dos dados pessoais tratados com base no consentimento;
- **Portabilidade:** solicitar a portabilidade dos dados a outro fornecedor de serviço;
- **Revogação do consentimento:** retirar o consentimento a qualquer momento, sem prejuízo do tratamento realizado anteriormente;
- **Oposição:** opor-se ao tratamento realizado com fundamento em outras bases legais.

Para exercer qualquer um desses direitos, entre em contato pelo e-mail: **marcusviniciusbp@gmail.com**

### 8. Privacidade de Crianças

O Aplicativo não é destinado a crianças menores de 13 anos e não coleta intencionalmente dados pessoais de menores. Caso tomemos conhecimento de que dados de uma criança foram coletados sem consentimento verificável dos pais ou responsáveis, tomaremos medidas para excluí-los imediatamente.

### 9. Segurança

Adotamos medidas técnicas e organizacionais para proteger os dados dos usuários contra acesso não autorizado, perda ou divulgação indevida, incluindo:

- Transmissão de dados criptografada via HTTPS/TLS;
- Controle de acesso ao Firebase Firestore via Firebase Security Rules;
- Autenticação obrigatória para acesso aos dados na nuvem.

Nenhum método de transmissão ou armazenamento eletrônico é 100% seguro. Em caso de incidente de segurança que afete seus dados, você será notificado conforme exigido pela LGPD.

### 10. Alterações a esta Política

Esta Política pode ser atualizada periodicamente. Quando houver alterações relevantes, o usuário será informado por meio de notificação no Aplicativo. A data da última atualização está indicada no topo deste documento. O uso continuado do Aplicativo após as alterações constitui aceitação da nova Política.

### 11. Contato e DPO

Para dúvidas, solicitações ou reclamações relacionadas a esta Política de Privacidade:

**E-mail:** marcusviniciusbp@gmail.com

| Purpose | Legal basis |
|---|---|
| Store and display expenses logged by the user | Performance of contract |

### 4. Data Storage

#### 4.1 Local storage
By default, all expense data and settings are stored exclusively on the user's device, using the operating system's internal storage. The App does not access the network by default.

When the user enables the "Cloud Sync" feature in settings, data is sent to and stored in Firebase Firestore (Google LLC), with servers located in the United States. This feature can be disabled at any time, and cloud data can be deleted upon request.

| Apple App Store / Google Play | Payment processing | Respective store policies |

### 6. Data Retention

- **Local data:** remains on the device until the user uninstalls the App or manually deletes the data.
- **Cloud data:** retained while the user's account is active. Can be deleted upon request to the contact email.
- **Analytics data:** retained for up to 14 months, in accordance with Firebase Analytics' default policy.

### 7. Your Rights

Depending on your jurisdiction, you may have the following rights regarding your personal data:

- **Withdrawal of consent:** withdraw consent at any time, without affecting processing carried out prior to withdrawal;
- **Objection:** object to processing carried out on other legal grounds.

The App is not intended for children under the age of 13 and does not knowingly collect personal data from minors. If we become aware that data from a child has been collected without verifiable parental consent, we will take steps to delete it immediately.
